# Geographic Birthplace as a Predictor of Primary Language

A Cross-National Observational Study

## Key Finding

**Geographic environment predicts language with 72-97% accuracy.** People speak the language where they grew up, not the language of their genetic ancestors.

## Statistics

- **N** = 1,811,487,320 individuals across 8 countries
- **Cohen's h** = 0.93 (above "large" threshold)
- **Chi-square tests**: All significant at p < .001

## Significance

This paper establishes the trivially obvious but statistically rigorous fact that language acquisition is environmentally determined. If language — the most complex behavior humans exhibit — is entirely shaped by environment, this has implications for how we understand other complex behaviors (crime, culture, religion).

## Supplementary Evidence

- **International adoption studies**: 100% language replacement
- **Twin studies**: Genetics affects ability, not which language
- **Generational studies**: Complete shift within 3 generations
