# What This Proves

## What we tested

We pulled together four completely independent lines of evidence and asked: if the criminal justice system actually finds truth, what would we expect to see? Then we checked what the data actually shows.

## What we found

Four pillars. All four go the wrong way.

**Pillar 1 — Lie detection:** If the system finds truth, people should be able to detect lies well above chance. Reality: 54%. Lie-specific detection: 47% (below chance). Training doesn't help.

**Pillar 2 — Memory:** If testimony reflects what happened, memory should be stable under questioning. Reality: a single misleading question after an event can alter someone's memory permanently. About 1 in 4 people will carry a materially changed memory into testimony and not know it. Average effect size d = 0.87 (that's large — it means memories shift almost a full standard deviation from a bit of language).

**Pillar 3 — False confessions:** If confessions are reliable, false confession rates should be negligible. Reality: 12-30% of exonerations involve false confessions. And exonerations only capture cases where someone survived long enough to prove it — the documented rate is a floor, not a ceiling.

**Pillar 4 — Suggestibility:** If confessions are voluntary, detention conditions shouldn't dramatically increase suggestibility (how easily someone can be led). Reality: arrest, sleep deprivation, isolation, and identity stripping each increase suggestibility by 31-56%. They compound. Conservative estimate: detention **doubles** your baseline suggestibility before the interrogation even starts.

## Why it matters

If even one of these pillars held up, you could argue the system works imperfectly but has a truth-finding core. None of them hold up. All four independently reject the claim. The data doesn't show a flawed truth-finding system — it shows a guilt-production system with truth-finding rhetoric.

## The numbers (simplified)

- **54%** lie detection (47% for lies specifically)
- **d = 0.87** memory distortion from misleading language (large effect)
- **12-30%** of exonerations involve false confessions
- **+80-120%** suggestibility increase from pre-interrogation detention alone
- **24,483** participants in the lie detection data, multiple independent sources for each pillar

## So what?

Four independent fields of research — deception detection, memory science, exoneration data, and suggestibility studies — all point the same direction. The system doesn't find truth. It produces guilt. The evidence isn't ambiguous. It's convergent.
