QUICK DECISION MATRIX

USE GEMINI IF YOU WANT:
✓ Exact customization (specific colors, text, layout)
✓ Specific data/numbers built in (like "91% inversion")
✓ Educational diagrams (arrows, flows, custom shapes)
✓ Control over content and presentation
✓ Original, unique designs

USE ADOBE STOCK IF YOU WANT:
✓ Fast, ready-to-use professional images
✓ Realistic photographs vs diagrams
✓ Guaranteed high quality/resolution
✓ Time-saving (no generation/editing)
✓ Professional publication-ready images
✓ Legal commercial use license included

RECOMMENDATION:
- Figures 1-3: GEMINI (flowcharts need custom layout)
- Figures 4-6: EITHER (timelines and charts work in both)
- Figures 7-11: ADOBE STOCK (easier to find good matches)

================================================================================
ADOBE STOCK TIPS FOR THESIS
================================================================================

1. Download with "License": Must include "Standard License" or "Extended License"
   (Standard is fine for thesis; Extended for publication/commercial use)

2. Resolution: Select "High Resolution" (300 DPI minimum for printing)

3. File format: Download as PNG or TIFF (preserves quality)

4. Attribution: Adobe Stock images in academic thesis should include:
   "Image from Adobe Stock" or photographer name if available

5. Custom licensing:
   - If image is close but not perfect, check if it can be licensed for editing
   - Some images allow minor text/label modifications

6. Collections:
   - Search "creative collections" for themed image sets
   - Often have matching visual styles across multiple images

7. Budget estimate:
   - Single images: AUD $10-30 each
   - Subscription (monthly): AUD $40-80 per month for unlimited downloads
   - For 11 images: ~AUD $25-50 per month subscription is often cheaper
