THESIS ARGUMENT KEY:
--------------------------------------------------------------------------------
If hedging_rate is HIGHER in truthful speech:
  => Innocent speakers qualify their statements (because memory IS uncertain).
  => Investigators/juries who read hedging as deception are constructing guilt
     from the linguistic behaviour of innocence itself.

If certainty_rate is HIGHER in deceptive speech:
  => Liars perform confidence. Rehearsed stories are clean and unqualified.
  => The Reid Technique's emphasis on linguistic confidence as a truth signal
     is systematically inverted.

If filler_rate is HIGHER in truthful speech:
  => Genuine recall is effortful and produces disfluency.
  => Disfluency is used as a deception cue by trained investigators.
  => Innocent cognitive effort is read as guilty concealment.
