# Language and Birthplace (Study Package)

Workspace for the geographic-language acquisition paper: `README_original.md` holds the full OMXUS markdown preprint (methods, nine-country census selection, chi-square and Cohen’s **h** / Cramér’s **V**, concordance **72–96.9%**). Subfolders include `latex/`, `publish/`, and `statistical_analysis/` for export, formatting, and replication alongside the parallel LaTeX repo at [../evidence_paradigm_shift_geographical_birthplace_predicts_language_aquisition/](../evidence_paradigm_shift_geographical_birthplace_predicts_language_aquisition/).

Use this tree when editing narrative and publication assets; use the evidence_paradigm_shift folder for the standalone TeX article and script-driven stats.

**OMXUS Goals:** **Goal 12** (accurate learning science) and **Goal 1** (democratic competence includes rejecting genetic determinism where evidence does not support it).
