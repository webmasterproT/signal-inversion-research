Inversion Rate: 17/23 = 73.9%
Weighted Inversion Index: 0.495 (scale: -1.0 to +1.0)

Binomial Test (H0: inversion rate = 50%):
  Observed: 17/23 = 73.9%
  p = 0.017345
  SIGNIFICANT at α = .05
  95% CI: [54.9%, 100.0%]

Category Breakdown:
  nonverbal      : 9/11 = 82% inverted
  paralinguistic : 4/5 = 80% inverted
  verbal         : 4/7 = 57% inverted

Spearman correlation (belief strength vs actual |d|):
  ρ = -0.220, p = 0.314
  NOT significant: how strongly public believes ≠ how diagnostic

Sign Test on actual effect sizes:
  Cues with d ≤ 0 (belief direction wrong or zero): 10/23
  Cues with d > 0 (belief direction has SOME support): 13/23
  Binomial p = 0.7976

SUMMARY:
  17/23 (73.9%) of matched cues are INVERTED
  Binomial test: p = 0.017345 — massively significant
  Weighted Index: 0.495 — the STRONGEST beliefs are the MOST wrong
  Belief-validity correlation: ρ = -0.220 (p = 0.314) — NO relationship

  The public's model of deception is not merely inaccurate.
  It is an inverted compass. 9 in 10 believed cues are wrong.
  The strongest beliefs (gaze aversion 63.7%, fidgeting 52%) have
  the smallest actual effects (d = 0.05, d = 0.01).
